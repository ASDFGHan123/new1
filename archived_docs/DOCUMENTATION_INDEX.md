## üöÄ Quick Commands

### Start All Services
```bash
# Windows
start-all.bat

# macOS/Linux
./start-all.sh
```

### Individual Services
```bash
# Django
python manage.py runserver --settings=offchat_backend.settings.development

# Redis
redis-server

# Celery
celery -A offchat_backend worker --beat -l info

# React
npm run dev
```
